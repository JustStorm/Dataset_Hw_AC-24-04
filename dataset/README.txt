Тут лежат датасеты. По очевидным причинам исходные датасеты на гитхабе не лежат, они весят 500Мб обработанный и 3Гб исходный.

US_Accidents_March23.csv   - Исходный, неизменный
Accidents_full.csv         - Исходный, обработанный

Остальные - создаются из исходного, по формату "Accidents_{num}k.csv" через скрипт parse_dataset.py в корне проекта.

Исходный датасет (US_Accidents_March23.csv) можно скачать по ссылке:
https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents/data
